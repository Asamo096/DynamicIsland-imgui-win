# DynamicIsland - 实现计划（分解和优先级排序的任务列表）

## [x] 任务 1：实现配置文件的实际读写功能
- **优先级**：P0
- **依赖**：None
- **描述**：
  - 实现 config.cpp 中的 Load 和 Save 方法
  - 支持 JSON 格式的配置文件读写
  - 确保配置更改在应用重启后保持
- **验收标准**：AC-5
- **测试要求**：
  - `programmatic` TR-1.1：修改配置后重启应用，验证配置是否保持
  - `programmatic` TR-1.2：检查配置文件是否正确生成和读取
- **注意**：使用现有的 SimpleJSON 类进行 JSON 解析

## [x] 任务 2：优化网络监控功能
- **优先级**：P1
- **依赖**：None
- **描述**：
  - 优化网络数据采集逻辑
  - 减少网络监控的系统开销
  - 提高网络速度计算的准确性

  - `programmatic` TR-2.2：检查网络监控对系统性能的影响
- **注意**：考虑使用更高效的网络数据采集方法

## [x] 任务 3：改进 GPU 监控对非 NVIDIA 显卡的支持
- **优先级**：P1
- **依赖**：None
- **描述**：
  - 为 AMD 和 Intel 显卡添加基本的监控支持
  - 通过 WMI 或其他通用方法获取 GPU 信息
  - 确保在没有 NVIDIA 显卡时仍能显示基本 GPU 信息
- **验收标准**：AC-1
- **测试要求**：
  - `programmatic` TR-3.1：在非 NVIDIA 显卡系统上验证 GPU 信息显示
  - `programmatic` TR-3.2：检查不同显卡厂商的兼容性
- **注意**：使用 WMI 或 DirectX 方法获取通用 GPU 信息

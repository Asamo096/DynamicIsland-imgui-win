# DynamicIsland - 状态栏菜单优化与设置系统规格文档

## 概述
- **摘要**：优化 DynamicIsland 状态栏的左键菜单功能，增加实用的快捷操作选项，并添加设置系统，通过独立的设置窗口配置灵动岛的各项功能。
- **目的**：提供更直观、更便捷的状态栏操作体验，同时为未来功能扩展提供配置管理框架。
- **目标用户**：使用 DynamicIsland 的 Windows 用户，需要频繁进行系统监控和快捷操作的用户。

## 背景与上下文
- DynamicIsland 是一个 Windows 系统监控工具，模仿苹果的灵动岛设计
- 当前状态栏功能相对简单，仅包含基础的显示信息
- 用户需要更丰富的快捷操作选项
- 需要一个统一的设置系统来管理日益增长的功能配置需求

## 目标
- 优化状态栏左键菜单，增加实用快捷操作
- 添加设置选项，点击后打开设置窗口
- 建立可扩展的设置系统框架
- 为未来功能预留配置接口

## 非目标
- 不实现具体的设置项功能（留空，待后续实现）
- 不修改现有的系统监控显示功能
- 不影响状态栏的正常显示性能

## 功能需求

### FR-1：状态栏左键菜单优化
- 分析当前状态栏菜单的使用场景
- 增加实用的快捷操作选项
- 确保菜单项符合用户日常使用习惯

### FR-1.1：当前菜单保留项
- 退出程序选项（Ctrl+Shift+Z）

### FR-1.2：新增快捷操作选项（示例）
- 显示/隐藏灵动岛
- 刷新系统信息
- 打开文件管理器（可选）
- 打开控制面板（可选）

### FR-2：设置系统框架
- 在状态栏菜单中添加"设置"选项
- 点击"设置"打开独立的设置窗口
- 设置窗口与主窗口分离，可独立操作
- 设置窗口采用模态或非模态对话框形式

### FR-2.1：设置窗口结构
- 设置窗口标题："DynamicIsland 设置"
- 设置窗口大小：600x400 像素（或适当尺寸）
- 设置窗口位置：屏幕居中
- 包含左侧分类导航（可选）和右侧内容区域

### FR-2.2：设置分类框架（预留）
- 通用设置（General）
- 外观设置（Appearance）
- 通知设置（Notifications）
- 文件中转站设置（Transfer Station）
- 高级设置（Advanced）
- 关于（About）

### FR-3：可扩展设置系统架构
- 使用配置管理类管理所有设置
- 支持设置的保存和加载
- 支持设置的默认值初始化
- 支持设置的实时应用（部分设置）

## 技术实现

### 配置管理
- 创建 `ConfigManager` 类或扩展现有 `Config` 类
- 支持从文件加载和保存设置
- 支持设置变化的监听和通知

### 设置窗口
- 使用 ImGui 创建独立的设置窗口
- 窗口独立于主渲染循环
- 支持键盘快捷键关闭设置窗口

### 状态栏菜单
- 使用 ImGui 的弹出菜单或原生窗口菜单
- 菜单项使用简洁的图标和文字
- 菜单响应迅速，无明显延迟

## 验收标准
- AC-1：状态栏左键菜单能够正常弹出
- AC-2：菜单中包含"设置"选项
- AC-3：点击"设置"选项能够打开设置窗口
- AC-4：设置窗口能够正常关闭
- AC-5：设置系统框架能够支持后续功能扩展
